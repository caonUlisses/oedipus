# Oedipus - Passing the Sphinx #

Oedipus is an auth service for micro-service systems, based on JWT and Node.JS, running on docker.

## How to use?

Clone the repo, cd in, run:

```
docker-compose build
```

wait for it... Then run:

```
docker-compose up
```
